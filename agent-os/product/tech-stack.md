# Tech Stack

## Runtime

- Node.js + TypeScript

## Parser

- Tree-Sitter (TypeScript grammar)

## Database

- PostgreSQL
  - **pgvector** — vector similarity search for code embeddings
  - **Apache AGE** — graph extension for dependency/relationship queries

## Embeddings

- Local LLM via Ollama or Llamafile

## MCP

- `@modelcontextprotocol/sdk` — MCP server SDK
